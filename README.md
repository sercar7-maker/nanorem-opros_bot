# Бот-опросник NANOREM

Telegram-бот для консультации клиентов по применению авто-продукции NANOREM.

## Возможности

- 📋 Интерактивный опрос для определения пригодности обработки NANOREM
- 🔧 Поддержка различных агрегатов: Двигатель, МКПП, АКПП, Редуктор (мост), ГУР
- 💰 Автоматический расчёт стоимости обработки
- 📝 Сохранение заявок в файлы (папка `applications/`)
- 📤 Отправка карточек клиентов администратору в Telegram

## Установка

1. Убедитесь, что у вас установлен Python 3.8 или выше:
   ```bash
   python --version
   ```

2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## Настройка

1. Получите токен бота у [@BotFather](https://t.me/BotFather) в Telegram

2. Создайте файл `.env` в корне проекта со следующим содержимым:
   ```
   BOT_TOKEN=ваш_токен_здесь
   RVS_PRICE_PER_ML=70
   ACCEL_PRICE_PER_ML=30
   MARKUP_COEF=2.0
   ```

   Или установите переменные окружения:
   ```bash
   # Windows PowerShell
   $env:BOT_TOKEN="ваш_токен_здесь"
   $env:RVS_PRICE_PER_ML="70"
   $env:ACCEL_PRICE_PER_ML="30"
   $env:MARKUP_COEF="2.0"
   
   # Windows CMD
   set BOT_TOKEN=ваш_токен_здесь
   set RVS_PRICE_PER_ML=70
   set ACCEL_PRICE_PER_ML=30
   set MARKUP_COEF=2.0
   
   # Linux/Mac
   export BOT_TOKEN="ваш_токен_здесь"
   export RVS_PRICE_PER_ML="70"
   export ACCEL_PRICE_PER_ML="30"
   export MARKUP_COEF="2.0"
   ```

   **Важно**: Все финансовые параметры загружаются из `.env` файла. Если переменные не установлены, используются значения по умолчанию.

3. Укажите ваш Telegram ID в файле `bot.py` (строка 10):
   ```python
   ADMIN_CHAT_ID = 899738024  # Замените на свой ID
   ```
   
   Чтобы узнать свой ID, напишите боту [@userinfobot](https://t.me/userinfobot) в Telegram.

4. (Опционально) Настройте финансовые параметры в файле `.env`:
   - `RVS_PRICE_PER_ML` - себестоимость РВС за 1 мл (по умолчанию: 70)
   - `ACCEL_PRICE_PER_ML` - себестоимость ускорителя за 1 мл (по умолчанию: 30)
   - `MARKUP_COEF` - коэффициент наценки для клиентской цены (по умолчанию: 2.0)
   
   Все параметры загружаются автоматически из `.env` файла при запуске бота.

## Запуск

```bash
python bot.py
```

Бот запустится и начнёт принимать сообщения. Для остановки нажмите `Ctrl+C`.

## Использование

1. Найдите вашего бота в Telegram
2. Отправьте команду `/start`
3. Следуйте инструкциям бота, отвечая на вопросы
4. В конце опроса вы получите рекомендацию по применению NANOREM
5. Администратор получит карточку с данными клиента и расчётом стоимости

## Команды бота

- `/start` - начать консультацию
- `/help` - показать справку
- `/cancel` - прервать текущую консультацию

## Формула расчёта стоимости

### Для двигателя:
- РВС: 10 мл на 1 литр рабочего объёма двигателя
- Ускоритель: 2.5 мл на 1 литр масла

### Для других агрегатов:
- Базовая ставка × объём масла × коэффициент агрегата

Коэффициенты можно настроить в файле `bot.py`.
